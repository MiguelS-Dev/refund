# 💸 Refund

Aplicação desenvolvida como desafio da **Rocketseat**, com o objetivo de praticar a manipulação do DOM utilizando **JavaScript**, permitindo registrar e gerenciar despesas para solicitação de reembolso de forma simples e intuitiva.

## 📖 Sobre o projeto

O **Refund** é uma aplicação que possibilita adicionar despesas informando o nome, categoria e valor. Conforme os itens são cadastrados, a lista é atualizada automaticamente, exibindo a quantidade de despesas registradas e o valor total do reembolso.

Este projeto foi desenvolvido para reforçar conceitos fundamentais de JavaScript, como manipulação de elementos HTML, captura de eventos, validação de formulários e atualização dinâmica da interface.

## ✨ Funcionalidades

- ➕ Adicionar novas despesas
- 📝 Informar nome, categoria e valor da despesa
- 📋 Exibir todas as despesas cadastradas
- ❌ Remover despesas da lista
- 💰 Calcular automaticamente o valor total
- 🔢 Atualizar a quantidade de despesas em tempo real
- 📱 Interface responsiva

## 🛠️ Tecnologias utilizadas

## 🚀 Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/MiguelS-Dev/refund.git
```

2. Acesse a pasta do projeto:

```bash
cd refund
```

3. Abra o arquivo `index.html` em seu navegador.

Ou utilize a extensão **Live Server** no Visual Studio Code para uma melhor experiência durante o desenvolvimento.
